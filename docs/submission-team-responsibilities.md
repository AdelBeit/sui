This was a solo project, so I was responsible for the full stack end to end: product design, UX, frontend and backend implementation in Next.js, the React renderer and chart components, the PocketBase data layer, and the server-side generation pipeline. For sponsor tools, I used Google DeepMind's Gemini API and A2UI protocol for the prompt-to-dashboard flow, LangChain for orchestration, and Daytona to run the PocketBase-backed sandbox environment and seed/setup workflow.
